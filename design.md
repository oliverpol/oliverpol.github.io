# Portfolio Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1a2332) - Professional, trustworthy, scientific
- **Secondary**: Warm Copper (#c17b5c) - Sophisticated accent, data visualization highlight
- **Tertiary**: Soft Sage (#8fa68e) - Calming, natural balance
- **Background**: Off-white (#fafafa) - Clean, readable
- **Text**: Charcoal (#2c3e50) - High contrast, professional

### Typography
- **Display Font**: "Tiempos Headline" - Bold, editorial serif for headings and hero text
- **Body Font**: "Suisse Int'l" - Clean, modern sans-serif for readability
- **Code Font**: "JetBrains Mono" - Technical content and code snippets

### Visual Language
- **Scientific Precision**: Clean lines, structured layouts reflecting analytical mindset
- **Data-Driven Aesthetics**: Subtle grid patterns, geometric elements suggesting statistical analysis
- **Professional Sophistication**: Minimalist approach with strategic use of whitespace
- **Biomedical Inspiration**: Organic curves balanced with technical precision

## Visual Effects

### Used Libraries
- **ECharts.js**: Interactive radar chart for skills visualization
- **Anime.js**: Smooth animations for timeline and hover effects
- **p5.js**: Generative background patterns mimicking data visualizations
- **Splide.js**: Smooth carousel for project showcase
- **Shader-park**: Subtle background effects suggesting data flow

### Effects & Animations
- **Hero Background**: Animated particle system representing data points in motion
- **Skills Radar**: Interactive chart with smooth transitions and hover details
- **Timeline**: Animated progression with staggered reveal animations
- **Project Cards**: 3D tilt effects on hover with depth shadows
- **Navigation**: Smooth scroll-triggered animations with easing

### Styling Approach
- **Grid System**: Precise alignment reflecting analytical thinking
- **Micro-interactions**: Subtle hover states and click feedback
- **Loading States**: Elegant skeleton screens for data-heavy sections
- **Responsive Design**: Fluid layouts maintaining visual hierarchy

### Header Effect
- **Animated Background**: Flowing data visualization with floating particles
- **Color Cycling**: Subtle color transitions in accent elements
- **Parallax Scrolling**: Gentle depth effects on scroll
- **Typography Animation**: Character-by-character reveal for key headings

This design creates a sophisticated, professional portfolio that reflects expertise in data science while maintaining visual appeal and usability.
